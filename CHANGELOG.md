# Change Log

All notable changes to the "PMD Code Checker" extension will be documented in this file.

## [1.0.0] - 2026-03-13

### Added
- Initial release of PMD Code Checker
- PMD7 ruleset support with custom configuration
- Git integration to detect pending files
- Webview panel for visual file selection and results display
- File check/uncheck functionality
- Run PMD check on selected files
- Generate Markdown reports with check results
- Auto-check on save option
- Configuration for PMD executable path
- Configuration for ruleset path
- Configuration for output report path
- Support for Java files
- Activity bar view container
- Tree view for pending files
- Command palette integration
- Keyboard shortcuts

### Features
- 🔍 PMD Code Checking with customizable rules
- 📁 Git Integration for pending file detection
- ✅ File Selection UI with checkboxes
- 📊 Visual Results panel with statistics
- 📝 Markdown Report Generation
- ⚙️ PMD7 Ruleset XML Format Support
- 🔄 Auto-check on Save option
